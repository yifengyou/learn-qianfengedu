## 算数运算符

```
+ - * / ** //
```

1. ** 是次方
2. // 是整除

![20200127_090654_12](image/20200127_090654_12.png)


\+ 运算

* 整型和浮点型就是数值相加
* 字符串类型就是字符串拼接

\* 运算

* 整型和浮点型就是数值相乘
* 字符串就是字符串重复

![20200127_094252_93](image/20200127_094252_93.png)

/ 运算

1. 除数不能为0，解释器运行时错误
